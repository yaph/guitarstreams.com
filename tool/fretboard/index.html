How to use this tool

Use this tool to show and learn the notes on the fretboard of the guitar and the ukulele. By default all 12 notes are shown, you can set your own in the "Notes to show" text field or select a scale such as C Major, A Mixolydian or E Phrygian Dominant. When selecting a diatonic scale the seven triads you can create from it show up below the fretboard. Clicking on a triad button highlights the corresponding notes with different colors for the root, third and fifth notes. For a quick demo, watch the video below.
